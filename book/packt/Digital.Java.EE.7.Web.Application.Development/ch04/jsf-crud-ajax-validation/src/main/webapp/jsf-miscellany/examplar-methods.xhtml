JSF Method Invocation Example

Method Invocation Examples

Invoke JSF controller with 2 dynamic arguments

\#{examplarController.methodTwoArgs( examplarController.city, examplarController.country )}

Invoke JSF controller with 3 literal arguments

\#{examplarController.methodThreeArgs('Obiwan','Ben','Kenobi')}

Invoke JSF controller method with parameters

\#{examplarController.methodPassingAttribute()
name="callToActionText" value="FindNearestDealer"
name="customerType" value="Motorbikes"

Invoke JSF controller method with an attribute

\#{examplarController.performAction()
name="contactName" value="Roy Keane"

Invoke JSF controller method f:setPropertyActionListener

\#{examplarController.performAction()
actionListener="\#{examplarController.attributeListener}"
name="contactName" value="Roy Keane"
