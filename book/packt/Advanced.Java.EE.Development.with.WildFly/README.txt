The code projects for the chapters are provided in the respective chapter folders. To import a project into Eclipse IDE select File>Import in Eclipse. In Import window select General>Existing Projects into Workspace. Click on Next. Select root directory with Browse button. Click on Finish. The Eclipse project gets imported into Eclipse IDE. 

Importing a project may throw some errors initially because the environment has not been set or the dependencies have not been downloaded initially. 

You may create a project from scratch, as per the instructions given in the chapter, and then copy the content of the code file/s into your project, if you wish.
